## Health app - React.js

### Development and test

Create databases for dev and test

Rename `env` to `.env` and add path to the GraphQL API.

###### Install dependencies

```
yarn
```

###### Run specs:

```
yarn test
```

###### Add new GraphQL types

Add the types in ./src/graphql and run:

```
yarn run codegen
```

###### Start the app

```
yarn start
```
